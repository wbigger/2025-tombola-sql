# Tombola SQL

Una tombola da usare a scuola a Natale per ripassare insieme le query SQL!

## Overview

Stampa le schede in [tombola-db.pdf](tombola-db.pdf), distribuiscile tra i partecipanti insieme a delle normali cartelle per tombola (3 righe con 5 numeri ognuna, da 1 a 90). La scheda con il database può essere stampata anche una per gruppo e riutilizzata.

Pensato per gli studenti del quinto anno del tecnico informatico o liceo scienze applicate.

## Getting Started

Apri il [github pages](https://wbigger.github.io/2025-tombola-sql/) di questo progetto e comincia a giocare!

Considerando che a dicembre di solito si è arrivati a studiare solo i comandi base di SQL, questo progetto usa solo `SELECT`, `WHERE`, e gli aggregatori base (`MIN()`,`MAX()`,`COUNT()`).

## Developer

Il repository contiene anche i file sql per la creazione delle tabelle e per verificare le query.
